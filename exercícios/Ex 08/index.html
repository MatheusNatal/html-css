Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica)

Nesta frase, tenho um termo em destaque usando a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (Não semântica)

Nesta frase, tenho um termo em ênfase usando a tag <em> (Semântica)

Texto Marcado

Nesta frase, temos um termo marcado aqui. Utilizando <mark>

Nesta frase, temos um termo marcado aqui.Com cor diferente, utilizando <mark style="background-color: ">

Há também uma terceira forma de utilizar, pondo:
<style>
mark {
background-color:;
}
À estrutura <head>.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno. Utilizando as tags <big>(Obsoleta) e <small>

Texto deletado

Podemos marcar um texto como exclúido para indicar que ele deve ser lido, mas não considerado. Utilizando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Utilizando a tag <ins> (Semântica)

Existe também o sublinhado com a tag <U> (Não Semântica)

Texto sobrescrito

Para inserir coisas do tipo: x20 + 3. Utilizando a tag <sup>

Texto subscrito

Para inserir coisas do tipo: H2O. Utilizando a tag <sub>
